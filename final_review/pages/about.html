AirIndia Traffic Dashboard

A web development project for monitoring real-time airway traffic across 5 major Indian airports. Built with vanilla HTML, CSS, and JavaScript — featuring ML-powered delay prediction, live flight maps, and unified data visualization.

🤖
k-NN ML Predictor
k-Nearest Neighbors algorithm trained on 25 data points — predicts flight delays using congestion, weather, wind, humidity, and time of day.
🗺️
Live Flight Map
Leaflet.js interactive map showing real airport positions, flight paths between airports, and live plane position markers.
📊
Chart.js Visualizations
11+ chart types across 6 pages — bar, line, doughnut, radar, stacked, and area charts with consistent dark theming.
🇮🇳
Real Indian Airport Data
Actual IATA codes, coordinates, capacities, and realistic traffic figures for DEL, BOM, BLR, MAA, and CCU airports.

📋 Disclaimer

This project was built purely for educational purposes as part of a web development course and is intended for academic demonstration only.

• All live flight data is sourced from the OpenSky Network, a public non-profit platform, and is used solely for academic use.
• This dashboard is not affiliated with, endorsed by, or connected to Air India, any airline, or any airport authority in any way.
• The AI delay predictions are generated by a simplified ML model and should not be used for actual flight planning or any real-world decision making.
• No commercial use is intended or permitted.
• Static weather and congestion data shown is simulated and does not represent actual operational data of any airport.
